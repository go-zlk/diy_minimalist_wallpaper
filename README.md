极简文字壁纸设计器 产品说明文档
1. 产品概述

极简文字壁纸设计器是一款专注于高审美、高效率的桌面壁纸定制工具。它打破了传统壁纸软件“下载即用”的限制，允许用户通过简单的交互，将个人格言、专注词汇（如 FOCUS, INFINITE PROGRESS）转化为极具设计感的 4K/高清壁纸。

2. 核心功能
2.1 实时创作系统

即时反馈渲染：采用双层异步渲染逻辑，用户调节文字内容、字号、颜色时，预览区域毫秒级实时更新。

坐标自由布局：支持在预览区直接点击/拖动，精确定位文字在屏幕上的显示位置。

2.2 高级视觉样式

无限叠影 (Infinite Shadow)：内置立体叠影算法，通过多层深度偏移绘制，模拟出极具冲击力的 3D 视觉感。

边缘修复斜体 (Smart Italic)：采用透明图层切变技术，修复了常规倾斜导致的背景裁剪和黑边问题。

字体智能映射：自动扫描 Windows 系统全量字体，支持自定义艺术字体加载。

2.3 专业色彩管理

Lightroom 风格调色盘：集成高精度 HSV 色环与 SV 坐标方块。

双维度编辑：支持“一键切换编辑目标”，通过同一个圆盘分别控制“文字色彩”与“背景色彩”。

数学级色彩对齐：通过坐标系校准，确保圆环视觉颜色与内部方块底色完全一致。

2.4 系统集成

一键壁纸设置：调用 Windows 底层 API (SystemParametersInfoW)，生成图片后立即设为系统壁纸。

3. 技术规格

编程语言：Python 3.12+

图形引擎：Pillow (PIL) —— 负责高分辨率图像合成与文字抗锯齿。

UI 框架：PyQt6 —— 负责响应式布局与高性能事件处理。

色彩模型：HSV (Hue, Saturation, Value) 浮点坐标映射。

4. 用户界面指南 (UX)

产品采用 “三步走”设计逻辑，侧边栏从上至下分为：

内容设计区：输入文本、选择字体、拉动滑块调节基础字号。

视觉色彩区：通过“字体”和“背景”切换按钮锁定编辑目标，然后在色轮上滑动取色。

艺术样式区：勾选功能开关，开启叠影或斜体特效。

操作区：蓝色醒目按钮负责执行壁纸应用。

5. 常见问题排查 (FAQ)

程序启动慢？

首次启动时系统正在扫描数千个字体文件，程序会优先显示窗口，稍后自动加载字体列表。

颜色选择不准？

已在最新版本中修复。请确保点击圆环外部选择色相（Hue），内部方块选择深浅。

斜体边缘有锯齿？

程序在应用壁纸时会生成高分辨率原图，预览时的轻微锯齿属于缩放显示优化，不影响最终壁纸质量。
